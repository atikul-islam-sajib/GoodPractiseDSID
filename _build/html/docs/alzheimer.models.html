A class for making predictions and computing various metrics on a given dataset.

This class is designed to handle prediction tasks, specifically for datasets related to Alzheimer’s disease. It includes methods for calculating accuracy, precision, recall, F1 score, generating a classification report, and computing a confusion matrix.

model_evaluation()[source]

Executes the prediction process on the test dataset using the pre-loaded model.

Precondition:

  • The model must be loaded and compatible with the test data format and dimensions.

  • The test_loader attribute should point to a valid data loader with test data.

Post condition:

  • The method returns a tuple containing arrays of predicted labels and actual labels.

Returns:

predictions (list of ndarray): The array of predicted labels for the entire test dataset. actual_labels (list of ndarray): The array of actual labels corresponding to the test dataset.

Return type:

tuple of (predictions, actual_labels)

Raises:

RuntimeError – If there is an issue in model prediction, possibly due to incompatible data format or model issues.

model_performance()[source]

Computes and returns key performance metrics of the model on the test dataset.

This method evaluates the model’s performance by calculating accuracy, precision, recall, and F1 score. It utilizes the Prediction class for generating predictions and computing metrics, and organizes the results into a Pandas DataFrame for easy analysis.

Returns:

A DataFrame with columns for Accuracy, Precision, Recall, and F1 score.

Return type:
